# Autonomous Mobile Robot (AMR) Navigation

## Features
- Simulated LiDAR sensor
- Occupancy Grid Mapping
- A* Pathfinding Algorithm
- Obstacle Detection
- Path Visualization

## Technologies
- Python
- NumPy
- Matplotlib

## Run

```bash
pip install -r requirements.txt
python main.py


Output
Green dot = Start
Red dot = Goal
Black = Obstacles
Blue line = Path


---

# 🚀 HOW THIS WORKS

### Step 1:
Robot scans environment (LiDAR simulation)

### Step 2:
Builds occupancy grid

### Step 3:
A* finds shortest path

### Step 4:
Graph shows navigation path
